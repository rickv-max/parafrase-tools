# 🚀 Parafrase AI - Web Tools Premium

Web Tools Parafrase AI super cerdas menggunakan React + Firebase dengan UI/UX modern, profesional, dan responsif di semua device (mobile, tablet, desktop).

## ✨ Fitur Utama

### 🤖 AI Parafrase Engine

- Powered by **Gemini Flash Latest API**
- 8 Mode Parafrase: Standard, Academic, Formal, Casual, SEO Friendly, Creative, Human Rewrite, Anti AI Detection
- 10 Mode Khusus Premium S2 (Teknik, Kedokteran, Ekonomi, Hukum, Pendidikan, Psikologi, Manajemen, Informatika, Akuntansi, Sosial & Politik)

### 🔐 Sistem Autentikasi

- Firebase Authentication
- Login Google
- Login GitHub
- Email & Password

### 💰 Sistem Berlangganan

- Plan Gratis (300 kata)
- Plan Pro (5000 kata)
- Plan Premium (Unlimited)
- Integrasi PakKasir Payment Gateway

### 📊 Dashboard User

- Riwayat parafrase lengkap
- Statistik penggunaan
- Status subscription
- Profile management

### 🎨 UI/UX Modern

- Dark Mode & Light Mode
- Glassmorphism Design
- Smooth Animations (Framer Motion)
- Responsive Grid Layout
- Mobile First Approach

## 🛠️ Tech Stack

### Frontend

- **React 18** - UI Library
- **Vite** - Build Tool
- **TailwindCSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Navigation
- **Zustand** - State Management

### Backend

- **Firebase Authentication** - User Auth
- **Firebase Firestore** - Database
- **Gemini Flash API** - AI Processing
- **PakKasir** - Payment Gateway
